The project was done fully by me. It is available to view through "logasenchant.com". The code for this website is available on my github page.

Sense Solutions

Sense Solutions was a team project that I undertook in Glasgow Caledonian University, alongside 4 other students.

Our project aimed to help people choose an outfite without having to get out of bed. The application created a table in a database for the user and allowed them to add, edit and remove entries.

The entries were articles of clothing that the user owned. Once the user finished inputing their articles of clothing into the application, the user was able to group the various entries and create outfits.

This project was based in Android Studio. However, I do not have access to the files.
